# Architecture v1.0

This directory contains the approved Architecture v1.0 source set produced under P0-B49.

The source set is assembled from the five approved normalized foundational packets under PG0-E001 and the corrected Architecture v1.0 release candidate. It contains public reference architecture only.

At this stage:

- the release source set is under controlled review;
- the immutable release commit and release tag are not yet active;
- PG0 remains pending formal closure against the immutable release;
- IKG0 is not granted;
- runtime/platform implementation coding remains prohibited.

The release boundary preserves the separation between Public Reference Architecture, Restricted Architecture, Private Authority, and Secrets.
