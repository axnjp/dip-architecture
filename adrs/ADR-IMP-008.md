# ADR-IMP-008 — Initial Private-Authority Source and Secret Posture

- Status: accepted and locked
- Classification: Public Reference Architecture
- Publication form: abstract public projection
- Complete decision body: Restricted Architecture

## Public decision

Protected authority and secret material remain outside public artifacts and are resolved only through governed interfaces; public documentation implies no real protected instance.

## Public consequences

Implementations must satisfy this decision without weakening the approved governing architecture, authority model, evidence posture, portability, or conformance obligations.

## Public verification

Conformance is evaluated through public requirements and restricted verification evidence. The public build requires no restricted source, protected configuration, or secret value.

## Disclosure boundary

Detailed topology, provider choices, protected paths, approval messages, restricted source identifiers, and evidence locators are retained outside the public build. This projection neither confirms nor denies the existence of any real protected instance.
