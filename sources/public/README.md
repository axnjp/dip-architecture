# Public Source Staging

Only source artifacts explicitly approved for `Public Reference Architecture` classification may be staged here.

The public build is allowlist-only. Restricted Architecture, Private Authority, and Secrets must never be copied here, loaded for redaction, or used as build inputs.

P0-B13 status: controlled public source `DIP-SRC-0001` has been admitted at `program-directives/p0-production-and-publication-governance-directive.md`. Its classification, approval state, source relationships, and integrity digest are recorded in the Source Catalog.

Admission of one controlled source does not authorize unlisted inputs. Every public-build source must remain explicitly allowlisted in `publication.manifest.json`.
