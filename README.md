# Domain Intelligence Platform Architecture

This repository is the canonical public architecture publication workspace for the Domain Intelligence Platform (DIP).

## Publication status

This is a draft P0 working corpus. Publication Gate PG0 has not passed, Architecture v1.0 has not been released, and implementation authorization IKG0 has not been granted.

This repository contains architecture and publication material only. It does not authorize or contain DIP application/runtime implementation.

## Public boundary

Only explicitly allowlisted Public Reference Architecture sources and derived public projections may enter this repository. Restricted Architecture evidence, Private Authority data, active private configuration, raw conversation exports, and secret values are excluded.

Public documentation may describe protected capability classes abstractly. It must not disclose whether any real protected instance exists.

## Working-revision notice

The current commit is a controlled working revision, not the final authoritative public source revision or an Architecture v1.0 release pin. Missing evidence is represented by explicit lifecycle and gap status; it is never reconstructed from assumption.

## Governance

Approved architecture and implementation decisions remain authoritative. Publication normalization preserves stable identifiers, classification boundaries, traceability, and lifecycle state. Detailed restricted approval locators remain outside the public build.
