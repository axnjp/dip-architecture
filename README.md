# Domain Intelligence Platform Architecture

This repository is the canonical public architecture publication workspace for the Domain Intelligence Platform (DIP).

## Publication status

Architecture publication production is in progress. Publication Gate PG0 has not passed, and implementation authorization IKG0 has not been granted.

This repository contains architecture and publication material only. It does not authorize or contain DIP application/runtime implementation.

## Public boundary

Only explicitly allowlisted Public Reference Architecture sources may enter the public build. Restricted Architecture, Private Authority data, active private configuration, and secret values are excluded.

Public documentation may describe protected capability classes abstractly. It must not disclose whether any real protected instance exists.

## Governance

Approved architecture and implementation decisions remain authoritative. Publication normalization preserves stable identifiers, approval evidence, classifications, traceability, and lifecycle state.
